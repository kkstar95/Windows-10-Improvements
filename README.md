# Windows 10 Improvements
Some QoL changes i use myself i wanted to share very likely compatible with Windows 11 too, even if its not compatible somehow dont worry its 100% reverseable. :ghost:

> [!NOTE]
🖨️ Print icon adds a small icon to print option so i wont accidently click when i right click any file.\
📘 Gedit option adds text/cmd/bat files to "Edit With Gedit" while keeping the Notepad "Edit" you need to install gedit for this.\
⏬ (https://download.gnome.org/binaries/win64/gedit/) OR (https://download.gnome.org/binaries/win32/gedit/2.30/) \
📖 Notepad option changes default "Edit" to "Edit With Notepad" and adds icon to make it simple.
> 

>[!CAUTION]
Do not press 1 twice when its active if you do you will get same menu twice, in order to fix this press 2 to remove and press 1 to add it again.\
YOU MUST USE "Run as Admin Mode" when you execute cmd/bat files here. (or when you create yourself)
